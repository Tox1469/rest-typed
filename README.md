# rest-typed

Gerador de cliente REST type-safe a partir de spec

## Instalacao

```bash
npm install rest-typed
```

## Uso

```ts
import * as lib from "rest-typed";
```

Veja `src/index.ts` para exemplos completos da API.

## API

Todos os exports publicos estao documentados em `src/index.ts`. O pacote e escrito em TypeScript e distribui declaracoes de tipos (`.d.ts`) junto do build.
